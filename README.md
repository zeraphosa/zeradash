## Launc application

```bash
  git clone https://github.com/zeraphosa/zeradash.git
```

- npm install
- npm start
- cd server and npm start

## Technologies

- React js, Scss, Axios, react-router-dom, Redux, i18next, react-quilljs
- Node js, Express, MySql, Nodemon, Postman, Cors
